\chapter{我是少年狼 I Was a Teenage Lycanthrope}

\begin{HauntStory}
	月光灑進房間，撫摸你的皮膚。你蹣跚步向窗戶並望向滿月。你的理性開始尖叫並死去，同時你的身體則因為新生的力量而欣喜地嚎叫⋯
\end{HauntStory}

\HauntSection{此刻}
\vspace*{-1em}
\begin{itemize}
	\item 你的探險者仍在遊戲內，但是變成了叛徒。
	\item 等同於玩家數量的\TokenMon{red}放置在一旁備用。
	\item 將一枚\TokenMon{red}放置在你的角色卡上，代表你現在變成了\Monster{Werewolf}。
	\item 將一枚\TokenMon{orange}（代表\Omen{Dog}）放進你所在的房間內。
	\item 丟下持有的所有物品及同伴卡牌，並隨之調整你的屬性。
	\item 將所有低於初始值的屬性重置為初始值。接著提升總和等同於英雄數量的屬性。例如，如果有三位英雄的話，你獲得三點來分配給你想增加的屬性（而不是所有屬性各加三級）。
\end{itemize}

\HauntSection{對於英雄你知道}
你原本可能知道些關於他們的事，但是現在你已經太狼了以至於無法回憶起。

\HauntSection{你獲勝於…}
…所有英雄皆死亡或變成\Monster*{Werewolf}時。

\HauntSection{你在你的回合必須…}
每個回合開始時，你獲得１點力量或１點速度。你無法將屬性提升超過最大值。

\vfill\null\pagebreak

\HauntMobStatusSection{\Omen{Dog}}
\HauntMobStatus{速度４\quad{}力量４\quad{}神志３}

你現在操控\Omen{Dog}（\TokenMon{orange}），忽視其預兆卡牌上的任何敘述，牠現在成為怪物並有特殊的規則。

\HauntSection{特殊攻擊規則}
\vspace*{-1em}
\begin{itemize}
	\item 你（\Monster*{Werewolf}）或\Omen*{Dog}擊敗英雄時，該角色如正常般受傷。在接下來的回合開始時，該英雄必須進行神志檢定 4+ 以抵抗詛咒，如果檢定失敗，他變成\Monster*{Werewolf}（將一枚\TokenMon{red}放置在他的角色卡上）且不再是英雄。他必須閱讀\strong{叛徒手冊}並立即執行\strong{此刻}章節的內容。
	\item 除非被\strong{生存指南}中提到的特殊物品攻擊，\Monster*{Werewolf}的任何屬性皆不會降至骷髏頭標誌。任何其他來源的傷害皆砍半（向上取整）。
	\item \Monster*{Werewolf}和\Omen*{Dog}皆無法攜帶任何物品及使用\Room{Mystic Elevator}。
	\item 在你獲勝時，若其他\Monster*{Werewolf}有成功殺掉英雄的話，操控該\Monster*{Werewolf}的玩家亦同時獲勝。
\end{itemize}

\HauntSection{如果你獲勝…}
\begin{HauntStory}
	你從屋內滑行至外面的庭院，享受你舌頭上那令人感到慰藉的血腥味。隨著你的尾巴輕彈，你躍過覆滿青苔的牆壁並在馬路旁邊落地。你已經可以嗅到一哩之內有更多的人類發出梣木般的汗水味。

	今晚的狩獵才剛開始而已。
\end{HauntStory}
