\chapter{漂浮之眼 The Floating Eye}

\begin{HauntStory}
  你的主人終於到來。你跪在地上，一顆巨大漂浮的眼睛從太空船上降下來，懸浮在你的上方。是時候了，該讓你的朋友們了解真相，並向他們的新領主俯首稱臣。
\end{HauntStory}

\HauntSection{此刻}
\vspace*{-1em}
\begin{itemize}
  \item 如果只有三或四名玩家，準備一枚\TokenMon{orange}（代表\Monster{Alien}）；如果有五或六名玩家，則準備兩枚。將這些標記放進作祟發生的房間裡。
  \item 將一枚\TokenPenta{Item}（代表\Monster{Spaceship}）放進作祟發生的房間裡。
  \item 將你的人物模型放置在\Monster*{Spaceship}上。你的探險者及持有的所有物品皆已經離開遊戲，等待太空船出發。
\end{itemize}

\HauntSection{對於英雄你知道}
他們正試圖阻止你綁架他們。他們有一個辦法可以從外星人的心靈控制中解放出來。

\HauntSection{你獲勝於…}
…所有英雄皆死亡或登上\Monster*{Spaceship}。

\vfill\null\pagebreak

\HauntMobStatusSection{\Monster{Alien}}
\HauntMobStatus{速度４\quad{}力量６\quad{}神志６}

\HauntSection{特殊攻擊規則}
\vspace*{-1em}
\begin{itemize}
  \item 除了進行力量攻擊以外，\Monster*{Alien}可以同時對所有在同房間的英雄使用神志攻擊以施展心靈控制。每一個攻擊皆分開處理（擲骰）。在此特殊攻擊下，如果英雄擊敗\Monster*{Alien}，則沒有人會受傷；如果\Monster*{Alien}擊敗英雄，該英雄不會受傷，但改為被\Monster*{Alien}控制。
  \item 當英雄被\Monster*{Alien}控制時，在他的回合中，你必須將他往\Monster*{Spaceship}移動。該英雄在在移動時無需為房間的效果做檢定或受傷。該英雄亦無法進行攻擊或任何其他行動。在該英雄進入\Monster*{Spaceship}所在的房間的下一個回合開始時，他將登上\Monster*{Spaceship}並離開遊戲。
  \item 英雄們知道某種秘密方法能將彼此從\Monster*{Alien}的心靈控制中解放出來。\Monster*{Alien}對每名英雄只能控制一次，當英雄被解放之後，他將永久免疫\Monster*{Alien}的心靈控制。
  \item \Monster*{Alien}對速度攻擊免疫（例如\Item{Revolver}及\Item{Dynamite}等）。
  \item 被震懾的\Monster*{Alien}仍然能維持他們對英雄們的心靈控制。
\end{itemize}

\HauntSection{如果你獲勝…}
\begin{HauntStory}
  你的主人對你送給他們的禮物很滿意。你的朋友將成為異種生物學實驗中擔任最優秀的實驗對象。此外，理所當然地，主人們承諾讓你保留那漂浮之眼作為紀念品。
\end{HauntStory}
