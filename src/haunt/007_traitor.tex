\chapter{嗜血常春藤 Carnivorous Ivy}

\begin{HauntStory}
	你一直都很喜歡植物。杜鵑花、紫苑、藤蔓⋯哦，對，藤蔓！你現在可以感受到藤蔓正爬上宅邸的外牆並滑進窗戶。

	你知道你必須幫助這些藤蔓不斷成長。植物需要什麼才能長得茂盛呢？陽光、水分⋯還有養分。什麼才能成為最好的肥料？啊！當然！你的朋友們！
\end{HauntStory}

\HauntSection{此刻}
\vspace*{-1em}
\begin{itemize}
	\item 你的探險者仍在遊戲內，但是變成了叛徒。
	\item 如果你攜帶著\Omen{Book}，必須將其丟下。你之後也不能再撿起它。
	\item 等同於玩家數量兩倍（至多十組）的\TokenMon{orange}（代表\Monster{Root}）及\TokenMon{green}（代表\Monster{Tip}）放置在一旁備用。請使用相同數字的標記代表相對應的\Monster*{Root}及\Monster*{Tip}，每組代表一株\Monster{Creeper}。
	\item 將\Monster*{Root}放進以下\textbf{\Monster*{Creeper}房間}內：\Room{Entrance Hall}、\Room{Balcony}、\Room{Bedroom}、\Room{Chapel}、\Room{Conservatory}、\Room{Dining Room}、\Room{Gardens}、\Room{Grand Staircase}、\Room{Graveyard}、\Room{Master Bedroom}、\Room{Patio}、\Room{Tower}。你不能在一間房間放置超過一個\Monster*{Root}。如果\Monster*{Creeper}房間的數量超過\Monster*{Root}的數量，你可以自行選擇要放在哪些房間。如果\Monster*{Root}的數量超過\Monster*{Creeper}房間的數量，你可以在新的\Monster*{Creeper}房間被探索到的時候放入追加的\Monster*{Root}標記。
	\item 在每個有\Monster*{Root}的房間放入對應數字的\Monster*{Tip}標記。
\end{itemize}

\HauntSection{對於英雄你知道}
他們正進行著某個計畫⋯某個能摧毀你珍愛藤蔓的計畫。那些英雄們會製造某個在\strong{生存指南}中提到的特殊物品。

\HauntSection{你獲勝於…}
…所有英雄皆死亡或該特殊物品被摧毀。為了摧毀這件物品，你必須先從持有他的英雄身上搶奪過來，接著到\Room{Chasm}、\Room{Furnace Room}、\Room{Underground Lake}將其丟掉已摧毀它。

\vfill\null\pagebreak

\HauntMobStatusSection{\Monster{Creeper}}
\HauntMobStatus{速度２\quad{}力量５\quad{}神志３}
\vspace*{-1em}
\begin{itemize}
	\item 你可以移動\Monster*{Tip}使\Monster*{Creeper}會成長，但\Monster*{Root}永遠不會離開它的起始房間。
	\item 如果\Monster*{Tip}進入了\Room{Mystic Elevator}，\Room*{Mystic Elevator}將停止運作直到\Monster*{Tip}離開。
\end{itemize}

\HauntSection{特殊攻擊規則}
\vspace*{-1em}
\begin{itemize}
	\item \Monster*{Root}無法移動，無法進行攻擊，也不會受到一般攻擊影響。但以上行為\Monster*{Tip}皆可以進行。
	\item 如果\Monster*{Tip}透過物理攻擊擊敗英雄，該英雄不會受傷，但改為被\Monster*{Tip}捕獲，並丟下持有的所有物品（落在該房間裡）。接著\Monster*{Tip}的回合立即結束。
	\item 其他\Monster*{Tip}無法攻擊被捕獲的英雄，但你可以。
	\item 在\Monster*{Creeper}的回合開始時，若\Monster*{Tip}捕獲著英雄，它必須往其對應的\Monster*{Root}移動兩格，而不會進行正常的移動。\Monster*{Tip}可以經由任何路徑回去\Monster*{Root}的位置。
	\item 在\Monster*{Creeper}的回合開始時，任何被捕獲回\Monster*{Root}的英雄將被殺害並成為肥料。將\Monster*{Creeper}（包含\Monster*{Root}及\Monster*{Tip}）移出遊戲。
	\item \Monster*{Root}不會妨礙英雄移動，但是\Monster*{Tip}會。
	\item \Item{Bell}對被捕獲的英雄沒有作用。\Omen{Spirit Board}對\Monster*{Tip}沒有作用。
\end{itemize}

\HauntSection{如果你獲勝…}
\begin{HauntStory}
	你躺在主臥室裡，看著藤蔓爬過你頭頂的天花板並滑過床單。現在屋內非常祥和。你很快就得再找些「朋友」來餵養你珍愛的藤蔓。

	你一向都是個園藝高手。
\end{HauntStory}
