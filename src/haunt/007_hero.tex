\chapter{嗜血常春藤 Carnivorous Ivy}

\begin{HauntStory}
  一道乾燥刺耳的聲音在屋內迴響。起初你確信是巨大的響尾蛇要來抓你，但你隨後便看見藤蔓正擠壓著窗戶。隨著玻璃破碎爆裂，你意識到那植物要過來了⋯向著你襲來。
\end{HauntStory}

\HauntSection{此刻}
將一枚\TokenPenta{Item}（代表\Monster{Plant Spray}）放置在一旁備用。

\HauntSection{對於壞人你知道}
\Monster{Creeper}會想辦法抓到你，屋內的每一組\Monster{Root}與\Monster{Tip}代表一株\Monster*{Creeper}。這些食肉植物以屍體製成的肥料為食。

\HauntSection{你獲勝於…}
…你使用\Monster*{Plant Spray}殺死等同於玩家數量的\Monster*{Creeper}時。

\HauntSection{如何製造\Monster*{Plant Spray}}
每回合一次，你可以在\Room{Research Laboratory}或\Room{Kitchen}內使用\Omen{Book}執行知識檢定達 5+ 嘗試製造\Monster*{Plant Spray}。如果檢定成功，拿取\Monster*{Plant Spray}標記。你只能製造一劑\Monster*{Plant Spray}，如果它被摧毀的話，就不能再重新製造了。

\vfill\null\pagebreak

\HauntSection{特殊攻擊規則}
\vspace*{-1em}
\begin{itemize}
  \item 當你攜帶\Monster*{Plant Spray}進入有\Monster*{Root}或\Monster*{Tip}的房間時，你可以對其噴灑\Monster*{Plant Spray}作為你這一回合的攻擊，如此將自動殺死該\Monster*{Creeper}。（\Monster*{Plant Spray}有足夠的劑量以除掉所有的\Monster*{Creeper}。）
  \item \Monster*{Root}無法進行攻擊，也不會受到一般攻擊影響，只有\Monster*{Plant Spray}對它有效。\Monster*{Tip}可以正常地進行攻擊及受到攻擊。
  \item 如果\Monster*{Tip}透過物理攻擊擊敗英雄，該英雄不會受傷，但改為被\Monster*{Tip}捕獲，並丟下持有的所有物品（落在該房間裡）。接著\Monster*{Tip}的回合立即結束。
  \item \Monster*{Root}不會妨礙英雄移動，但是\Monster*{Tip}會。
\end{itemize}

\HauntSection{你在你的回合必須…}
如果你被\Monster*{Tip}捕獲，你無法使用物品，但你依然可以對\Monster*{Tip}進行物理攻擊嘗試掙脫。若你擊敗\Monster*{Tip}，它將被震懾並放開你，接著你便可以正常進行接下來的回合。如果你被\Monster*{Tip}擊敗，你依然不會受到傷害，但你的回合會立即結束。

\HauntSection{如果你獲勝…}
\begin{HauntStory}
  藤蔓痛苦地扭動並揮打，砸碎了花瓶、繪畫和傢俱。那一瞬間，藤蔓感覺好像要把房子震垮了一般，但它們隨後就溜走了，縮回到地面之下。你現在只聽見哭泣聲。但是誰在哭呢？噢⋯原來是你自己。
\end{HauntStory}
